📋 Setup Instructions

Step 1 — Create the Google Sheet

Open sheets.google.com, create a new sheet named Stock Entry Data.

In Row 1, add these headers exactly:

Branch | Date | Item | S1_1PM | S2_7PM | S3_10PM | Timestamp

Step 2 — Add the Apps Script

In the sheet: Extensions → Apps Script. Delete all code and paste this:

function doPost(e) {
var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
var data = JSON.parse(e.postData.contents);
data.rows.forEach(function(row) { sheet.appendRow(row); });
var output = ContentService.createTextOutput(JSON.stringify({status:"ok"}));
output.setMimeType(ContentService.MimeType.JSON);
return output;
}
function doGet(e) {
var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
var values = sheet.getDataRange().getValues();
var output = ContentService.createTextOutput(JSON.stringify(values));
output.setMimeType(ContentService.MimeType.JSON);
return output;
}

Step 3 — Deploy

Click Deploy → New Deployment

• Type: Web App

• Execute as: Me

• Who has access: Anyone

Click Deploy → Copy the Web App URL → Paste it above.

Step 4 — Host on GitHub Pages (free)

1. Create a free account at github.com

2. Create a new repository (name it anything, e.g. stock-entry), set to Public

3. Upload this file renamed as index.html

4. Go to Settings → Pages → Deploy from branch → main → Save

5. Your site will be live at: https://yourusername.github.io/stock-entry/

✓ All branches can now open this URL and submit entries. All data saves to your Google Sheet!
